## Обзор

Инструмент, созданный для эффективного и чистого извлечения текста из веб-ресурсов. В отличие от обычных скраперов на базе браузеров, которые потребляют значительные ресурсы системы, Srappy использует мощь Rust и легковесность Tauri. Это позволяет получить высокую скорость работы без лишней нагрузки на компьютер.

Приложение ориентировано на создание качественных датасетов для машинного обучения, проведения исследований и архивации контента, где чистота текста и сохранение его структуры имеют решающее значение.

## Ключевые возможности

### Движок "чистой" экстракции

Использует алгоритм рекурсивного обхода дерева элементов для выявления смысловых блоков, отсекая интерактивные элементы, рекламу и навигационный мусор.

### Рекурсивный обход (Crawling)

Встроенная поддержка глубокого парсинга. Укажите начальную точку и максимальную глубину — движок автоматически построит карту внутренних ссылок и пройдет по ним, соблюдая границы домена и ваши фильтры.

### Поддержка современного веба

Основной движок работает максимально быстро через прямые HTTP-запросы. Однако для сайтов, требующих выполнения JavaScript (React, Vue, Angular), в предусмотрена опциональная интеграция с Headless Chrome.

### Защита от блокировок

Для стабильного доступа к данным поддерживает:

- Ступенчатые задержки запросов (каскадные таймеры).
- Автоматическую ротацию User-Agent.
- Управление пулом прокси с рандомным выбором.

## Аналитическая панель

Каждая сессия скрапинга — это не просто скачивание файлов, а статистическое событие. Srappy предоставляет продвинутый набор инструментов для мониторинга:

- **Живые метрики:** Отслеживание скорости парсинга (страниц в секунду) и объема данных в реальном времени.
- **Визуализация трендов:** Интегрированные графики на кривых Безье позволяют наглядно оценить эффективность работы на протяжении всей истории.
- **Детальные отчеты:** Каждая запись в истории снабжена глубокой аналитикой: плотность данных (символов на байт), примерное время чтения контента и соотношение успешных запросов к ошибкам.

## Техническая архитектура

- **Бэкенд:** Rust, использование `tokio` для асинхронного управления задачами и `reqwest` для сетевых операций.
- **Фронтенд:** Svelte 5, обеспечивающий реактивный интерфейс с минимальными задержками, который остается отзывчивым даже при обработке тысяч логов.
- **Взаимодействие:** Tauri bridge для безопасной и типизированной связи между UI и системным уровнем.
